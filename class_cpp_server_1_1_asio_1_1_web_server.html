Detailed Description

Web server.

Web server is used to provide HTTP/HTTPS interface to handle different kind of Web requests such as POST, GET, PUT, DELETE, etc.

Thread-safe.

https://github.com/corvusoft/restbed

Examples:
web_http_server.cpp, and web_https_server.cpp.
