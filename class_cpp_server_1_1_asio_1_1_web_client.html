Detailed Description

Web client.

Web client is used to send different kind of Web requests such as POST, GET, PUT, DELETE, etc. to any HTTP/HTTPS Web server and receive responses in synchronous and asynchronous modes.

Thread-safe.

https://github.com/corvusoft/restbed
