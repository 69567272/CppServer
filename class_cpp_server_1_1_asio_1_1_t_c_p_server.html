Detailed Description

template<class TServer, class TSession>
class CppServer::Asio::TCPServer< TServer, TSession >

TCP server.

TCP server is used to connect, disconnect and manage TCP sessions. It is implemented based on Asio C++ Library and use a separate thread to perform all TCP communications.

Not thread-safe.

Examples:
tcp_chat_server.cpp.
