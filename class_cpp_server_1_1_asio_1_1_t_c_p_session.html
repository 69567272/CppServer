template<class TServer, class TSession>
class CppServer::Asio::TCPSession< TServer, TSession >

TCP session.

TCP session is used to read and write data from the connected TCP client.

Not thread-safe.

Handle buffer received notification.

Notification is called when another chunk of buffer was received from the client.

Default behavior is to handle all bytes from the received buffer. If you want to wait for some more bytes from the client return the size of the buffer you want to keep until another chunk is received.

Parameters
buffer- Received buffer
size- Received buffer size
Returns
Count of handled bytes
